✔️ To-Do List Bot

The To-Do List Bot allows you to manage a to-do list via Discord!

Adding the Bot

To add the bot to your server, click this button!

Commands

To use the bot, type any of the following commands:

td! add (date) [item]
This command adds the item you specify to your to-do list on the date you specify, if you specify a date.
td! done [id]
This command marks the item you specify as complete.
td! rm [id]
This command removes the item you specify from your to-do list.
td! list (date)
This command lists all items on your to-do list. Optionally, you can specify a date.
