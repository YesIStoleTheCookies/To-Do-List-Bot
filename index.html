To-Do List Bot

The To-Do List Bot allows you to manage a to-do list via Discord!

Adding the Bot

To add the bot to your server, simply click this button!

Commands

To use the bot, simply type any of the following commands:

td!add [item]
This command adds the item you specify to your to-do list.
td!done [item]
This command marks the item you specify as complete.
td!rm [item]
This command removes the item you specify from your to-do list.
td!list
This command lists all items on your to-do list.
